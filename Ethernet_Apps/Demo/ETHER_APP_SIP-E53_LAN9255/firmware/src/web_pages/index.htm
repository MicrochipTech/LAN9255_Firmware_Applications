On the right you'll see the current status of the demo board. For a quick example, click the GPIO out to toggle the GPIO pin on the board. Change the input of GPIO Pins, you'll see the status update immediately. This examples uses AJAX techniques to provide real-time feedback.

This site is provided as a tutorial for the various features of the HTTP web server, including:

  • Dynamic Variable Substitution - display real-time data
  • Network configuration - require a user name and password. User Name: admin       Password: microchip
  • Uploading Files - parse files for configuration settings and more

This example application has support for updating configuration parameters, controlling GPIO pin, calculate md5sum and configuring network interface.Thanks to built-in GZIP compression support, this examples fit in the 32kB of on-board Memory.
